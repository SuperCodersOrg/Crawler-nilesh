# Daily Journal – 17 July 2026



## Objective

* Understand computer networking fundamentals.
* Learn how TCP and UDP protocols work.
* Research the design of an Indexer for the web crawler.
* Improve page fetching by integrating both cURL and CDP.

**Work Done:**

* Watched two networking videos provided by the mentor:

  * Studied how networking works from client to server.
  * Learned the working of TCP and UDP protocols and their differences.
* Researched the architecture and working of an Indexer for the crawler project, focusing on word frequency counting and inverted indexing.
* Enhanced the crawler by integrating **cURL** along with the existing **CDP-based** page fetching mechanism.
* Implemented a mechanism to determine whether a webpage is static or dynamic before fetching.
* Created the `isDynamicPage()` function, which:

  * Detects common JavaScript-only placeholders such as "You need to enable JavaScript".
  * Removes `<script>`, `<style>`, and HTML comments from the downloaded HTML.
  * Calculates the amount of visible text remaining after removing HTML tags.
  * Classifies pages with very little visible content (less than 200 characters) as dynamic pages.
* Updated the crawler workflow:

  * First attempts to fetch the page using **cURL**.
  * If the page is detected as dynamic, automatically fetches the fully rendered HTML using **Chrome DevTools Protocol (CDP)**.
  * Static pages continue to use the faster cURL-based fetching.

**Problems Faced:**

* Identifying dynamic webpages reliably without unnecessarily launching Chrome for every request.
* Designing an efficient approach that balances performance and correctness for both static and JavaScript-rendered pages.

**What I Learned:**

* Fundamental concepts of computer networking.
* Differences between TCP and UDP communication.
* Basic design principles of an Indexer and inverted indexing.
* A hybrid fetching strategy (cURL + CDP) provides better performance than relying solely on CDP while still supporting JavaScript-heavy websites.

**Outcome:**

* Successfully integrated both cURL and CDP into the crawler.
* Implemented automatic static/dynamic page detection.
* Improved crawler efficiency by using cURL for static pages and CDP only when required.
* Gained a clear understanding of networking concepts and the indexing pipeline for the next phase of the project.
